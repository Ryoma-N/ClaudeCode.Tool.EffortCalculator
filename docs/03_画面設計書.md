## SCR-01 ダッシュボード

### 概要

ログイン後の起点。全プロジェクトの概況・アラート・統計を表示。

### 画面レイアウト

```
┌──────────────────────────────────────────┐
│ ダッシュボード          2026年3月17日(火) │
├──────────────────────────────────────────┤
│ [🔴 期限超過 2件]                         │ ← AlertSection
│   • ログイン機能実装  3日超過             │
│ [🟡 3日以内に期限 1件]                    │
├──────────────────────────────────────────┤
│ 総工数(h)  実績(h)  進捗中  完了         │ ← OverallStats
│   240h      80h       3      1           │
├──────────────────────────────────────────┤
│ 進行中プロジェクト                        │
│ ┌──────────────┐ ┌──────────────┐        │
│ │ ECサイト改修  │ │ API開発      │        │ ← ProjectSummaryCard
│ │ ████░░ 65%   │ │ ██░░░░ 30%   │        │
│ │ 見積80h 実績52h│ │ 見積40h 実績12h│       │
│ └──────────────┘ └──────────────┘        │
└──────────────────────────────────────────┘
```

### 表示要素

| 要素 | 内容 |
|---|---|
| 日付 | 本日の日付（曜日付き） |
| アラートセクション | 期限超過タスク（赤）・3日以内期限タスク（黄）の一覧。クリックでタスク詳細へ |
| 全体統計 | 全プロジェクトの見積合計・実績合計・進行中プロジェクト数・完了プロジェクト数 |
| プロジェクトカード | 進行中プロジェクトのみ表示。進捗バー・見積/実績を表示 |

## SCR-03 プロジェクト詳細

### 概要

WBS（フェーズ・タスク一覧）と進捗サマリーの確認。タスク・フェーズの追加。

### 画面レイアウト

```
┌──────────────────────────────────────────┐
│ ← ECサイト改修              ✏️ ⚙️ 📄 🗑️ │
│    2026/01/01 〜 2026/03/31              │
├──────────────────────────────────────────┤
│ [進捗サマリー]                            │
│ 見積工数  出来高    実績工数              │
│  120h     78h       82h                  │
│ ──────────────────────────────           │
│ 進捗率  ██████░░░░ 65%                   │
│ 工数消化率 ████████░░ 68%                │
│ 乖離: +4.0h (+0.5人日)                   │
├──────────────────────────────────────────┤
│ [フェーズ別棒グラフ] [工程別円グラフ]    │
├──────────────────────────────────────────┤
│ ▼ 設計フェーズ                ✏️ 🗑️     │
│   進捗率 100% 見積20h 出来高20h 実績18h  │
│   ─────────────────────────────────      │
│   ✅ 基本設計書作成  山田   100%  20h/20h│
│   ✅ DB設計          佐藤   100%  18h/20h│
│                                          │
│ ▶ 開発フェーズ                ✏️ 🗑️     │
│                                          │
│ フェーズなし                             │
│   ○ テスト環境構築   田中  50%  0h/8h   │
│                                          │
│ [＋フェーズ追加] [＋タスク追加]          │
└──────────────────────────────────────────┘
```

### 進捗サマリーの要素

| 要素 | 説明 |
|---|---|
| 見積工数 | 全タスクの見積合計（h / 人日） |
| 出来高 | Σ（見積 × 進捗率）（h / 人日） |
| 実績工数 | 全WorkEntryの合計（h / 人日） |
| 進捗率バー | 出来高ベースの進捗（紫→黄→緑） |
| 工数消化率バー | 実績/見積（参考値。赤くなると予算超過） |
| 乖離 | 実績 − 出来高（+ = 予算超過傾向） |

### タスク行の表示要素

| 要素 | 説明 |
|---|---|
| ステータスバッジ | 未着手（灰）・進行中（青）・完了（緑） |
| タスク名 | クリックでタスク詳細へ |
| 担当者 | タスクに設定された担当者名 |
| 進捗率 | 手動設定された% |
| 見積/実績 | `実績h / 見積h` 形式 |
| 進捗ミニバー | 進捗率を視覚化 |

## SCR-05 タスク詳細

### 概要

タスクの詳細情報確認・進捗率更新・日次作業実績の入力。

### 画面レイアウト

```
┌──────────────────────────────────────────┐
│ ← ECサイト改修                 ✏️ 🗑️    │
│ ログイン機能実装                          │
│ 認証機能のフロントエンド実装              │
│ [進行中] [2026/03/31] [👤 山田 太郎]     │
├──────────────────────────────────────────┤
│ 進捗率                            60%   │
│ ██████░░░░░░░░░░░░░░ (バー)              │
│                                          │
│  見積工数      出来高      実績工数       │
│   16h          9.6h        10h           │
│  2.00人日    1.20人日    1.25人日        │
│ ──────────────────────────────────       │
│ 乖離（実績 − 出来高）    +0.4h (+0.05人日)│
├──────────────────────────────────────────┤
│ 工程別実績                    合計 10h   │
│ ● 設計    4h                            │
│ ● 実装    6h                            │
│ ─────────────────────────────────        │
│ 新規実績入力                             │
│ 日付 [2026-03-17]  工程 [実装   ▼]       │
│ 時間 [____h]  備考 [__________________]  │
│ [＋ 記録を追加]                          │
│ ─────────────────────────────────        │
│ 実績履歴                                 │
│ 2026-03-17 実装  3h  コーディング        │
│ 2026-03-16 実装  2h                      │
│ 2026-03-15 設計  4h  詳細設計完了        │
└──────────────────────────────────────────┘
```

### EVM指標の説明

| 指標 | 説明 |
|---|---|
| 進捗率 | 手動で設定する達成度（0-100%） |
| 出来高 | 見積工数 × 進捗率 ÷ 100（本来かかるべき時間） |
| 実績工数 | 実際に作業した時間の合計 |
| 乖離 | 実績 − 出来高（+ = 計画より工数がかかっている） |

### 日次実績入力フォーム

| 項目 | 入力形式 | 必須 | 説明 |
|---|---|---|---|
| 日付 | 日付ピッカー | ○ | デフォルト本日 |
| 工程 | セレクト | ○ | プロジェクトで定義した工程から選択 |
| 時間 | 数値（0.5刻み） | ○ | 作業時間(h) |
| 備考 | テキスト | — | 作業内容メモ |

### タスク編集モーダル

| 項目 | 入力形式 | 必須 |
|---|---|---|
| タスク名 | テキスト | ○ |
| 説明 | テキストエリア | — |
| フェーズ | セレクト | — |
| ステータス | セレクト（未着手/進行中/完了） | ○ |
| 期日 | 日付 | — |
| 見積工数(h) | 数値（0.5刻み） | — |
| 進捗率 | スライダー（0-100%、5刻み） | ○ |
| 担当者 | テキスト | — |

## SCR-07 月次レポート

### 概要

対象月・プロジェクトで絞り込んだ工数集計。担当者別・工程別集計と全明細を表示。PDF出力。

### 画面レイアウト

```
┌──────────────────────────────────────────┐
│ 月次レポート              [▼ PDF出力]   │
├──────────────────────────────────────────┤
│ [2026-03 ▾] [全プロジェクト ▾]           │ ← フィルター
├──────────────────────────────────────────┤
│ 合計工数   人月換算   稼働日数  記録件数 │
│  82h        0.51人月   18日     24件     │
├──────────────────────────────────────────┤
│ 担当者別              工程別             │
│ ─────────             ──────             │
│ 山田 太郎  40h 0.25人月 設計 20h 0.13人月│
│ 佐藤 花子  30h 0.19人月 実装 45h 0.28人月│
│ 田中 一郎  12h 0.08人月 テスト17h 0.11人月│
├──────────────────────────────────────────┤
│ 作業明細                                 │
│ 日付     PJ        タスク  工程 担当者 h  │
│ 03/17  ECサイト改修 認証実装 実装 山田 3h │
│ 03/17  API開発     設計    設計 佐藤 2h  │
│ ...                                      │
│                      合計      82h       │
└──────────────────────────────────────────┘
```

### フィルター

| 項目 | 入力形式 | 説明 |
|---|---|---|
| 対象月 | 月ピッカー（YYYY-MM） | デフォルト当月 |
| プロジェクト | セレクト | 全プロジェクト / 個別選択 |

### サマリーカード

| カード | 計算方法 |
|---|---|
| 合計工数(h) | 対象期間・プロジェクトの全WorkEntry合計 |
| 人月換算 | 合計工数 ÷ (mmDays × mdHours) |
| 稼働日数 | ユニークなrecordedDateの数 |
| 記録件数 | WorkEntryの件数 |
